# tlk-ui

## Purpose
React + TypeScript user interface for clients, stylists, and admins.

## Responsibilities
- Public marketing pages (pricing, stylists, portfolio)
- Client portal (profile, booking, payments, reviews)
- Stylist portal (availability, appointments, portfolio)
- Admin portal (manage stylists/users)
